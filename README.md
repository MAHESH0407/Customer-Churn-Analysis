# Customer Churn Analysis

## Project Overview

Customer churn is one of the biggest challenges for subscription-based businesses. This project analyzes customer behavior, identifies the factors contributing to customer churn, predicts whether a customer is likely to leave, and presents business insights through an interactive Power BI dashboard.

---

## Problem Statement

Businesses lose revenue when customers discontinue their services. The objective of this project is to analyze customer data, identify important churn indicators, and build a predictive solution that helps organizations improve customer retention.

---

## Objectives

- Analyze customer demographics and behavior
- Identify key factors influencing customer churn
- Predict customer churn using Machine Learning
- Visualize important KPIs using Power BI
- Generate actionable business insights
